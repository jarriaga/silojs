Silo Events

Silojs supports the following events:

  • click
  • dblclick
  • mousedown
  • mouseup

You can add silo events to dom elements with the silo-[eventName] attribute

<script type="text/javascript"> function testClick(){ alert('This is a silo-click test'); } </script> <button silo-click="testClick">Silo Click</button>

When working with nested controllers and views, it is important to provide the full class name and function name as the silo-event value. The following would be an example of how you would add a click event to an external view.

<silo:controller> <a href="#" silo-click="Controller.Test.testClick">Silo Click</a> </silo:controller>
