Persist Data Binding with Silojs

Silo.Bind Method

Silojs provides binding functionality to dom elements. To bind each loops do this

<silo:each products as="item" persist> <h3>{{item.title}}</h3> <p>{{item.description}}</p> </silo:each>

By providing the persist attribute to the silo:each element any time the products collection changes the silo element is re-rendered. This will ensure persistent syncronization of the dom element and data source.

You can also use persist data bining on form element. The example above will listen for change on Controller.Product.item and automatically update the input's value.

<input type="text" silo-bind="Controller.Product.item">
